# Strafe-NB-Keyboard
A keyboard for nohboard made custom to the roblox game: Strafe

## Steps:
1. Download the "Installer" zip [here](https://github.com/RyGuy-12/Strafe-NB-Keyboard/releases/download/1/Installer.zip)
2. exctract the folder
3. In the folder, run "KeyboardInstaller.bat"
4. If all is correct, you should have my custom strafe keyboard in NohBoard!

## Information about the installer:
* It looks for NohBoard-ReWrite-v1.3.0 (newest version)
* The nohboard folder must be in: (tell me if you want another path to be added)
  1. Downloads
  2. Documents
  3. OneDrive/Documents
